# Editor Role Map

## L8 role
Host the editor shell, route interaction, expose panels and view surfaces, and translate user intent into legal lower-stack requests.

## L9 role
Provide domain-specific authoring experiences and suite composition above canonical lower-stack families.

## L10 role
Provide import/export, graph, automation, scripting, project bootstrap, plugin, and packaging services.

## L11 role
Provide collaboration, review, approval, playtest, reporting, and onboarding surfaces.

## Never roles
- no `editor/` layer owns lower-stack transaction authority
- no `editor/` layer owns raw engine runtime
- no `editor/` layer may create a hidden parallel world state
