# Editor Visual Composition Model

## Canonical frame
- top menu bar
- top primary toolbar
- central tabbed workspace
- one or more viewport panes
- left navigation zones
- right inspector/details zones
- bottom diagnostics and output zones
- status bar
- command palette overlay
- contextual overlay bars

## Canonical panel set
- content browser
- outliner
- world browser / partition browser
- details
- asset inspector
- material inspector
- diagnostics log
- profiler / budgets
- validation
- build/release center
- assistant dock
- task monitor
- source and change review surface

## Layout rule
All panels are dockable, pinnable, hideable, saveable per workspace, and restorable.
No panel may become mandatory frame-hot if it is not visible.
