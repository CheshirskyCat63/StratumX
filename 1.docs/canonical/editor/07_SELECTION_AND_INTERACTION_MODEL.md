# Selection and Interaction Model

## Selection classes
- entity selection
- actor/object selection
- component selection
- asset selection
- terrain region selection
- spline/path selection
- fracture cluster selection
- partition cell selection
- graph node selection
- diagnostics item selection

## Interaction requirements
- single select
- multi-select
- lasso/marquee where legal
- hierarchy selection
- pivot editing
- drag/drop between browsers and views
- contextual right-click actions
- property edit staging
- batch edit
- duplicate / align / distribute / group
- isolate / lock / mute / hide

## Interaction law
Selection truth remains below `L8`.
`L8` hosts interaction state and focused tool context only.
