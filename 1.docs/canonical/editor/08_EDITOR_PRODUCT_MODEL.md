# Editor Product Model

The editor product is a coherent Rust-native creation environment comparable in breadth to a modern AAA editor.
It is viewport-centric, panel-rich, suite-extensible, assistant-assisted, validation-aware, and production-capable.

## Core product pillars
- one shell, many workspaces
- viewport-first world interaction
- universal tool contexts and overlays
- strong panel/view composition
- broad domain suites above one shared editor core
- service-backed asset and automation workflows
- production operations without truth leakage

## Mandatory product surfaces
- shell frame
- one or more viewports
- outliner/world browser
- content browser
- details/inspector
- command palette and shortcut system
- diagnostics surface
- assistant surface
- build/release surface

## Product rule
The product may be broad, but it must remain activation-bounded and resource-aware.
