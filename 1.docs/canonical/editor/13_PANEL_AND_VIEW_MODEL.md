# Panel and View Model

## Mandatory panels
- viewport host
- outliner/world browser
- content browser
- details/inspector
- diagnostics
- assistant
- build/release

## Optional panels
- graph editor
- validation results
- task/batch center
- plugin manager
- compare/diff
- review/annotation browser

## Panel law
Panels are hosts for projections, requests, and job states.
Panels do not own authoritative domain state.
