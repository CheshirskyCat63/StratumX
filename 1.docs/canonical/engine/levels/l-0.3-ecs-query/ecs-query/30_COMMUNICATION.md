# Communication

## Inputs

Registry truth, access model, identity, and handles.

## Outputs

Query descriptors, traversal descriptors, filters, joins, and read/write access descriptors.

## Canonical Data Forms

- query descriptors
- filters
- joins
- access descriptors

## Upward Flow

`engine_ecs_query` communicates upward through the listed canonical data forms rather than by hidden ownership transfer.
