# Realtime Outputs

## Role

Output contract for realtime runtime.

## Output Ownership Matrix

| Output class | Canonical owner | Ceiling source | Failure posture | Illegal posture |
|---|---|---|---|---|
| presentable frame | runtime realtime shell | queue + visibility + profile law | degrade decorative presentation first | queued-frame reservoir |
| audio submission | acoustics in runtime-owned publication window | acoustics ceilings | shed ambience/detail first | hidden audio-side frame pacing |
| remote publication | `L2.5` in runtime-owned publication window | network ceilings | shed noncritical publication first | presentation-driven network widening |
| replay/checkpoint markers | replay law in bounded windows | replay ceilings | delay optional compare first | opportunistic frame-path checkpointing |
| diagnostics readback | diagnostics law only | diagnostics ceilings | drop diagnostics first | stealth readback growth |

## Output Failure Matrix

| Pressure source | Mandatory preservation | First degradation |
|---|---|---|
| queue pressure | presentable frame legality | decorative quality |
| network pressure | authoritative publication windows | noncritical publication |
| replay pressure | cadence legality | compare/report extras |
| diagnostics pressure | none | diagnostics first |

## Illegal Patterns

- realtime outputs taking ownership of world truth, transport state, or residency governance;
- output-driven queue widening;
- hidden multi-frame buffering.

## Output Window Matrix

| Window class | Required owner | Illegal owner |
|---|---|---|
| presentable frame publication | runtime realtime shell | imaging-owned publish loop |
| remote publication window | runtime-owned network window | presentation-driven network flush |
| audio submission window | runtime-owned output slot | acoustics-owned cadence governor |

## Local Operating Law

Realtime outputs remain bounded publications.
They may not hide queue debt, shift authority ownership, or create multi-frame shadow buffers.
