# Society

## Role

Bounded social/economy interaction solve.

## Society Matrix

| Concern | Canonical law | Failure posture | Illegal posture |
|---|---|---|---|
| economy/social cadence | slower bounded cadence | defer low-value updates first | combat-rate full society solve |
| long-range pressure | approximate/event-driven only | collapse far aggregates first | always-on dense global society solve |
| network export | shaped bounded records only | trim optional exports first | direct transport ownership |

## Rule

Society runs on slower bounded cadences and shaped exports only. It may not silently become a combat-rate always-on global simulation.

## Cadence Matrix

| Cadence class | Canonical posture | Illegal posture |

| slower social/economy cadence | legal | combat-rate full society solve |
| long-range aggregate updates | bounded and approximate | dense global always-on updates |

## Local Operating Law

Society is intentionally slow, approximate, and bounded.
Any local rule that normalizes combat-rate global society work is illegal.
