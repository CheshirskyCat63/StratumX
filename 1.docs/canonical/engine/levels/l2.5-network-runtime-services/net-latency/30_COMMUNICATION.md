# Communication

## Inputs

Owned-input history, authoritative snapshots, transport timings, and runtime legality windows.

## Outputs

Predicted buffers, reconcile results, rewind-validation products, correction products, and latency diagnostics.

## Canonical Data Forms

- descriptors
- request batches
- staged/exported products
- diagnostics

## Upward Flow

`engine_net_latency` communicates upward through the listed canonical data forms rather than by hidden ownership transfer.
