# Communication

## Inputs

Structured generation context, model-backed or rule-backed generation jobs, and runtime-issued work windows.

## Outputs

Generation products, packaged outputs, transformation products, and diagnostics.

## Canonical Data Forms

- generation jobs
- generation products
- transformed outputs
- packaged outputs

## Upward Flow

`engine_generation` communicates upward through the listed canonical data forms rather than by hidden ownership transfer.
