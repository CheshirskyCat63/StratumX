# Resource Products

## Role

Prepared resource products.

## Canonical Definition

`Resource Products` is a canonical element of `engine_content` inside `L3.2. Resource Systems`. It exists as one explicit part of the engine stack and does not transfer its ownership class into another crate or level.

## Data Model

Prepared resource products are deterministic, versioned, and startup-bindable.
Product compatibility is explicit and does not create upward crate dependencies.
