# Glossary

ingress packet: canonical write envelope toward public engine L4.

binding control: control that changes association, scope, or attachment without carrying bulk mutation payload.

execution signal: control that triggers bounded runtime action such as start, stop, pause, rebuild, or refresh.

observation record: bounded upward publication of status or event data.

metric frame: bounded upward publication of counters, timings, pressure, or budget consumption.

compatibility fact: immutable exported fact used for admissibility decisions.

compatibility verdict: derived decision computed from compatibility facts.

legality verdict: derived decision stating whether a mutation class is legal.

opaque handle: scoped token for live engine-side entities or services.

opaque ref: scoped read-side token for immutable or monotonic projections.

opaque artifact ref: scoped read-side token for deterministic generated products.
