# StratumX SDK L5 Constitution

L5 exists to provide a narrow, typed runtime truth bridge above engine L4 and below editor/tooling L6.

L5 must remain:
- typed
- narrow
- non-UI
- non-editor
- non-product-semantic
- non-monolithic

L5 owns only boundary envelopes, compatibility facts, compatibility verdicts, transport policies, legality gates, and opaque runtime-facing handles and refs.
