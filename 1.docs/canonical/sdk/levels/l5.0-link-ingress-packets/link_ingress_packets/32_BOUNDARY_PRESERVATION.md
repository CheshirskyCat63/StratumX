# Boundary Preservation

## Adjacent above
L6 request publishers only

## Adjacent below
compat facts, transport policies, legality gates, runtime handles, public L4 packet surfaces

## This layer must not become
- an adjacent semantic class
- editor orchestration
- engine execution logic
- payload-body ownership outside its local contract

## Why the boundary matters
`link_ingress_packets` would lose semantic sharpness if it collapsed into an adjacent layer.
