# Boundary Preservation

## Adjacent above
L6 readers, object-handle readers, and state-ref readers only

## Adjacent below
public L4 identity export surfaces only

## This layer must not become
- an adjacent semantic class
- editor orchestration
- engine execution logic
- payload-body ownership outside its local contract

## Why the boundary matters
`engine_identity_refs` would lose semantic sharpness if it collapsed into an adjacent layer.
