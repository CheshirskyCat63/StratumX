# Document Rules

Every layer or family directory must contain:
- `00_LAYER.md`
- `10_LIBRARIES.md`
- `20_DEPENDENCIES.md`
- `30_COMMUNICATION.md`
- `31_THREADING.md`
- `32_BOUNDARY_PRESERVATION.md`
- `40_FIELDS.md`

Every `00_LEVEL.md` must state:
- why the level exists;
- what authority it owns;
- what it explicitly does not own;
- which adjacent collapse it forbids.

Each `40_FIELDS.md` must state for each field:
- name;
- semantic type;
- required or optional;
- invariant;
- ownership note.

Generic placeholder text is forbidden.
