# L7 Studio Orchestration Model

`L7` is the studio orchestrator.
It is cold, compiled-control, and never part of ordinary hot editor mutation.
It owns campaigns, workflow graphs, governance-bound runs, automation policy, and reporting.

## L7 owns
- compiled project orchestration graphs
- content, world, simulation, and release campaign bundles
- automation meta and schedules
- governance policies and freeze gates
- reporting requests and reporting views

## L7 never owns
- frame-level editor runtime
- editor mutation authority
- direct engine writes
- assistant reasoning truth
- raw mutable mirrors of `L6`

## L7 exchange law
`L7` may only address `L6` through compiled campaign bundles, task bundles, governance policies, automation requests, and reporting requests/results.
