# StratumX Tools Stack Constitution

The upper tools stack is frozen as four non-overlapping roles:
- `L6` editor authority and operating runtime
- `L6A` assistant runtime bridge
- `L7` studio orchestration and governance
- `L7A` assistant planning and reasoning

Only `L6` owns editor mutation authority.
All upper-layer exchanges are typed, bounded, and budget-visible.
Any hidden truth store, shortcut path, or role leakage is unconstitutional.
