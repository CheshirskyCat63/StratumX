# Family Level

Canonical family: `material_response_authoring_family`

## Composes
- material response authoring, reaction tables, and response rule authoring

## Data responsibility
- authority-facing minimal truth: material-response edit intents
- snapshot classes: response snapshots
- index classes: response lookup indices
- derived classes: derived response views
- artifact classes: response artifacts and manifests
- preview classes: material response previews
- cache classes: response caches
- diagnostics classes: response diagnostics
- degradation priority: `medium`

This family composes canonical planes and sidecars without owning hidden truth.
