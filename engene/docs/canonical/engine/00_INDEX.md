# StratumX Canonical Engine Stack

## Purpose

This package is the unified canonical engine stack for StratumX.

It contains:

- fixed lower substrate;
- shared world property substrate;
- runtime family;
- critical simulation families;
- service layers;
- startup;
- constitutional laws;
- implementation-facing build order and starter contracts.
