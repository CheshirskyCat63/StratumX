# Communication

## Inputs

World snapshots, ECS query surfaces, region context, and runtime execution windows.

## Outputs

Navigation products, perception products, decision intents, society deltas, and schedule products.

## Canonical Data Forms

- navigation products
- perception products
- action intents
- society deltas
- schedule products

## Upward Flow

`engine_agents` communicates upward through the listed canonical data forms rather than by hidden ownership transfer.
