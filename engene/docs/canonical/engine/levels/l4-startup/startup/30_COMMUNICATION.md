# Communication

## Inputs

Configuration, profile selection directives, world assembly inputs, and family registration inputs.

## Outputs

Launched runtime instance, wired family set, startup diagnostics, and initialized execution environment.

## Canonical Data Forms

- config
- profile selection
- runtime instance
- startup diagnostics

## Upward Flow

`engine_startup` communicates upward through the listed canonical data forms rather than by hidden ownership transfer.
